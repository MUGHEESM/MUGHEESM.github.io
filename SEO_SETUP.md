### Step 1: Verify Your Site

1. Go to: **https://search.google.com/search-console**
2. Click **"Add Property"**
3. Enter: `https://mugheesm.github.io`
4. Choose verification method:
   - **HTML file upload** (easiest for GitHub Pages)
   - Download the HTML file
   - Upload it to your repository root
   - Push to GitHub
   - Click "Verify"

### Step 2: Submit Sitemap

1. In Search Console, go to **"Sitemaps"** (left sidebar)
2. Enter: `sitemap.xml`
3. Click **"Submit"**
4. Google will start crawling your site!

### Step 3: Request Indexing

1. Go to **"URL Inspection"** (top bar)
2. Enter your homepage: `https://mugheesm.github.io/`
3. Click **"Request Indexing"**
4. Repeat for:
   - `https://mugheesm.github.io/blog.html`
   - `https://mugheesm.github.io/Walthroughs/Blue/blue-walkthrough.html`
